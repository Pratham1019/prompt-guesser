# Prompt Guesser Backend Services

This directory contains the production-ready Python backend services for **Prompt Guesser**, a daily multiplayer reverse-engineering prompt game.

---

## 🛠️ Technology Stack & Infrastructure

1. **API Engine:** FastAPI (Python 3.13)
2. **Database:** Supabase PostgreSQL
3. **ORM:** SQLAlchemy (Asyncio, asyncpg)
4. **Migrations:** Alembic
5. **Storage Layer:** Supabase Storage (Public bucket: `daily-challenges`)
6. **Task Scheduler:** Deterministic lookahead Daily Challenge scheduler
7. **Semantic Evaluation:** Gemini API (`gemini-3.1-flash-lite`)
8. **Image Generation:** Hugging Face serverless Inference Provider (`FLUX.1-schnell`)

---

## 🚀 Database Migrations

Database schemas are managed using **Alembic**.

### Apply Migrations to Supabase:
```bash
uv run alembic upgrade head
```

### Create a New Migration:
When models are updated, generate a new autogenerated migration script:
```bash
uv run alembic revision --autogenerate -m "description_of_changes"
```

## 📂 Supabase Storage Bucket Setup

1. In your Supabase project, navigate to **Storage**.
2. Create a new bucket named **`daily-challenges`**.
3. Toggle the bucket visibility to **Public** (required to serve generated daily images to the frontend).
4. Uploaded daily challenges are saved deterministically using the path structure:
   `daily/YYYY/MM/YYYY-MM-DD.jpg`

---

## 🧹 Expired Challenge Cleanup

The backend includes a dedicated **`CleanupService`** to remove historical challenge data and their associated Supabase storage images.

* Run the manual cleanup CLI command:
  ```bash
  python manage.py cleanup
  ```
* In production, this command is run automatically via a daily cron schedule to enforce your data retention policy (default is `30` days, configurable via `SCHEDULER_RETENTION_DAYS`).

## 🧪 Running Integration Tests

Integration scenario tests can be run using a local SQLite file to bypass network requirements during development.

```bash
# Run gameplay flows scenario test
DATABASE_URL="sqlite+aiosqlite:///./test_temp.db" TZ="UTC" uv run python test_gameplay.py

# Run scheduler lookahead & publication test
DATABASE_URL="sqlite+aiosqlite:///./test_temp.db" TZ="UTC" uv run python test_scheduler.py

# Run semantic judge evaluation test
DATABASE_URL="sqlite+aiosqlite:///./test_temp.db" TZ="UTC" uv run python test_evaluator.py
```
